install NestJS:
1- install NodeJS:
	- insure you have it already or not:
		node -v
		npm -v

	- if not:
		Go to: https://nodejs.org
		Download LTS (Long-Term Support) version for Windows.
		Run the installer → keep default options → finish.

2- install NestJS (nest cli):
	- run this command on the terminal:
		npm install -g @nestjs/cli

	- verify:
		nest --version
